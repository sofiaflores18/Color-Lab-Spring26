# LAMP Stack Application

## Description
COLORS is a web application built using the LAMP stack that allows users to log in and manage color records through a browser-based interface connected to a PHP/MySQL backend API.

## Technologies Used
- Linux
- Apache
- MySQL
- PHP
- HTML
- CSS
- JavaScript

## Project Structure
colors-lamp/
  api/        # PHP backend endpoints
  public/     # Frontend files served by Apache

## Setup Instructions
1. Install a LAMP environment (Linux/Apache/MySQL/PHP) or use XAMPP.
2. Create the required MySQL database and tables.
3. Update database credentials in the API files (placeholders are provided).
4. Place the project folder in your web server directory (e.g., htdocs or /var/www/html).
5. Access the application through:
   http://localhost/colors-lamp/public/

## Assumptions / Limitations
- Database must be configured manually.
- Credentials are placeholders and must be replaced locally.
- Minimal error handling (lab scope).
